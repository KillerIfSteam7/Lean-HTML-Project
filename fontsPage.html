CSS Fonts

As You See The Website Font Is Set To
Alef-Regular Font Which Isn't The Default Font
So, You Can Change Your Website Font Using
CSS @font-family Function, @font-family Is A
Function Which Defines A Set Of Fonts Or A Single
Font The Tag [e.x. header tag] Main Font As The
Tag's Main Font, If The Family Is Multiple Fonts
The Browser Searchs For In It Until It Finds
A Font That It Supports It Hover, If The Browser
Didn't Find A Supported One It Switchs Back To
Arial.
